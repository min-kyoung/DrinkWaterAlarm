# DrinkWaterAlarm
## Description
Local Notification을 이용해서 물마시기 알람을 구현하는 프로그램이다. <br>
언제 물을 마실지 시간을 정하고 해당 시간을 알림을 보낼 시간으로 설정한 후, 추가한 알림을 켜고 끌 수 있도록 한다.<br>
